## About

A full-featured social network platform where users can create profiles, share posts, follow other users, and interact through a real-time feed.

## Features

- 👤 **User Profiles** — Customizable profiles with avatars
- 📝 **Posts & Feed** — Create posts, like, comment
- 👥 **Follow System** — Follow users, personalized feed
- 📸 **Media Upload** — Share images and files
- 🔐 **Authentication** — Secure login and registration
- 🗄️ **Prisma ORM** — Type-safe database operations

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Runtime | Node.js |
| Framework | Express.js |
| Database | Prisma ORM |
| Language | JavaScript |
| File Storage | Local uploads |

## Getting Started

```bash
git clone https://github.com/foxnaim/Network-Social.git
cd Network-Social
yarn install
yarn start
```

## Project Structure

```
├── controllers/  # Route handlers
├── routes/       # API endpoints
├── prisma/       # Database schema
├── uploads/      # User uploaded files
└── app.js        # Entry point
```
